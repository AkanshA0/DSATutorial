Arrays

Arrays store items in contiguous memory and allow O(1) random access by index.

Common operations

  • Read by index - O(1)
  • Insert/delete at end - amortized O(1)
  • Insert/delete at start/middle - O(n)
  • Search (linear) - O(n) or binary search - O(log n) on sorted arrays

Example: Two Sum across languages

javascript 
function twoSum(nums, target) {
  const seen = new Map();
  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];
    if (seen.has(complement)) return [seen.get(complement), i];
    seen.set(nums[i], i);
  }
  return null;
}

python 
def two_sum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return None

java 
import java.util.HashMap;
import java.util.Map;

public class TwoSum {
  public static int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> seen = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
      int complement = target - nums[i];
      if (seen.containsKey(complement)) {
        return new int[]{ seen.get(complement), i };
      }
      seen.put(nums[i], i);
    }
    return null;
  }
}
